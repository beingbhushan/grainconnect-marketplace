# GrainConnect

GrainConnect is a frontend prototype of a grain marketplace where farmers can directly connect with customers in cities like Pune and Mumbai.

The idea came from observing how farmers often depend on middlemen and don’t get fair prices. This project explores a simple UI approach to make listings and buying more direct and transparent.

## What’s built so far

- Basic marketplace UI for grain listings
- Product cards and layout for browsing
- Initial structure for farmer-side interaction
- Responsive design (still improving)

## Tech used

- React
- JavaScript
- SCSS
- Vite

## Current status

The frontend is mostly in place. Backend integration (Node.js, MongoDB) and payment features are planned but not completed yet.

## What I plan to improve

- Improve UI and filtering options
